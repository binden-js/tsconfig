Shareable [TypeScript](https://www.typescriptlang.org/) config.

## Installation

```sh
npm i --save-dev @binden/tsconfig
```

## Usage

- Add the `extends` property (in the `tsconfig.json` file)

```json
{ "extends": "@binden/tsconfig" }
```
